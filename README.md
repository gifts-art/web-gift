## Стартиране

Отвори `index.html` в браузър или публикувай папката в GitHub Pages.

## Структура

```
web-gift/
├── index.html
├── styles.css
├── script.js
├── images/          # 6 снимки за галерията
└── README.md
```

## Мъжки вариант

Идентичен шаблон за мъжки получател: `../web-gift-him/` (демо: получател **Алекс**, подател **Мария**).

## Персонализация за клиент

1. Смени името в `.hero__name`
2. Обнови писмото в `#letterOverlay`
3. Смени снимките в `images/` и пътищата в галерията
4. Редактирай въпросите в `script.js` → `quizData`
5. Промени тайната изненада в `.scratch__prize-text`
